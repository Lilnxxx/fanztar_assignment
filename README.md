
# fanztar Assignment

This is a node api which using POST to send data
* The API takes a json format components array with proper codes
* The API searches for all specific codes and return proper cost and list of items


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
one can test the api using postman or other alternatives by using the link "Locahost:8800/api/id"
and putting data in the json area of body which has only one attributes named as components (array)




## Running Tests

To test the api ,the test cases file has been provided which contains random tests strings to check the working of the api
